## Description ##

Allows disabling Reftagger on a per-page/post basis

## Installation ##

### Manual Installation ###

1. Upload the entire `/reftagger-toggle` directory to the `/wp-content/plugins/` directory.
2. Activate RefTagger Toggle through the 'Plugins' menu in WordPress.
